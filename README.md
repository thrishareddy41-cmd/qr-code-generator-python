# QR Code Generator using Python

## Project Overview
This is a simple and user-friendly **QR Code Generator Desktop Application** built using **Python Tkinter**.

The application allows users to:
- Enter any text or URL
- Generate a QR code instantly
- Preview the QR code inside the application
- Download the QR code as a PNG image

This project is beginner-friendly and demonstrates how GUI applications can be built using Python.

---

## Features
✅ Generate QR codes from text or URLs  
✅ Supports up to **50,000 characters** input  
✅ Automatically splits very large input into multiple QR images  
✅ Live QR code preview inside the application  
✅ Download QR code as PNG  
✅ Automatically saves generated files  
✅ Simple and clean graphical user interface (GUI)

## Installation

### Step 1: Clone the repository
```bash
git clone https://github.com/your-username/qr-code-generator.git
cd qr-code-generator
```

### Step 2: Install required libraries
```bash
pip install qrcode[pil]
```

### Step 3: Run the project
```bash
python main.py
```

---

## How to Use
1. Open the application.
2. Enter text or a website URL.
3. Click **Generate + Download QR**.
4. The QR code will be generated and displayed on the screen.
5. It will automatically save in the project folder.
6. Click **Download PNG** to save it to any location.
